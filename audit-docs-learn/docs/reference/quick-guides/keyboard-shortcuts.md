# ⌨️ Keyboard Shortcuts

This page lists reliable shortcuts that improve speed while working in CloudAudit Pro and the Learning Hub.

Because the in-app shortcuts can vary by browser, OS, and UI framework, treat the “Product/UI” items below as *patterns* to confirm in your deployed UI.

## Universal (Windows/browser)

These work in most browsers and web apps:

- Find on page: `Ctrl+F`
- Next match / previous match: `Enter` / `Shift+Enter`
- New tab / close tab: `Ctrl+T` / `Ctrl+W`
- Refresh: `Ctrl+R`
- Open link in new tab: `Ctrl+Click`
- Jump to address bar: `Ctrl+L`
- Zoom in/out/reset: `Ctrl++` / `Ctrl+-` / `Ctrl+0`

## Learning Hub (Docusaurus)

- Search (most Docusaurus sites): `Ctrl+K`
- Page outline navigation: use the right-side “On this page” (mouse-driven, but fast)

If search isn’t enabled in your environment, it may depend on Algolia configuration.

## Product/UI patterns to look for

Many enterprise audit tools provide shortcuts like these:

- Global search / command palette: commonly `Ctrl+K`
- Quick navigation (jump to module): commonly `Ctrl+G` or `Ctrl+P`
- Save draft / save changes: commonly `Ctrl+S`
- Add comment / note: commonly `Ctrl+Enter` (when a comment box is focused)

## How to discover the real in-app shortcuts

1) Look for a “Keyboard shortcuts” item in the user menu or Help menu.
2) Check tooltips on buttons (some UIs show shortcuts on hover).
3) If a command palette exists (often `Ctrl+K`), open it and search “shortcuts”.
